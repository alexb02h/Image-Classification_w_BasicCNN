# CIFAR-10 CNN Image Classifier

## How to Run

### 1. Clone the repository
```bash
git clone https://github.com/alexb02h/CSCI_611_Alexander_Baron.git
cd CSCI-611_Alexander_Baron
```

### 2. Install dependencies
```bash
pip install torch torchvision matplotlib numpy
```

### 3. Open the notebook
```bash
jupyter notebook
```
Then open the `ipynb` file from the Jupyter interface in your browser. Or if you have Visual Studio Code, you can just open the .ipynb file directly from the interface.

### 4. Run the noteback

The CIFAR-10 dataset is downlaoded automatically on the first run via `torchvision.datasets.CIFAR10`. No manual download is required.

Run all cells in order using **Kernel -> Restart & Run All**
